# Accessibility Improvements for Bad Eyesight

This document outlines the specific improvements made to maximize readability for users with vision difficulties while maintaining MTG card size constraints (2.5" × 3.5").

## Typography Enhancements

### Font Size Optimization
- **Card Titles**: Increased to 11px screen / 9pt print (was 8px/6pt)
- **Card Subtitles**: Increased to 8px screen / 7pt print (was 6px/5pt) 
- **Body Text**: Increased to 8px screen / 6.5pt print (was 7px/5.5pt)
- **Cost/Badge Elements**: Increased to 7px screen / 6pt print (was 6px/5pt)
- **Stats Box Content**: Increased to 7px screen / 6pt print (was 6px/5pt)

### Font Weight & Style
- **All body text**: Changed to font-weight: 500 (medium weight) for better visibility
- **Card subtitles**: Made bold (font-weight: 600) instead of just italic
- **Key information**: Made bold throughout
- **Print text**: All text made font-weight: 600 or bold for printing

## Color & Contrast Improvements

### High Contrast Colors
- **All text**: Changed to pure black (#000) instead of gray variations
- **Card borders**: Changed to solid black instead of light gray (#ddd)
- **Background elements**: Darker backgrounds for better contrast
- **Stats boxes**: Added black borders for clear definition

### Badge/Marker Colors
Made all badge colors darker for better print visibility:
- **Passive markers**: #1a5d1a (darker green)
- **Skill markers**: #b8860b (darker gold) with white text
- **Reaction markers**: #cc5500 (darker orange)
- **Condition badges**: #4a2c7a (darker purple)
- **Action elements**: #003d7a (darker blue)
- **Cost elements**: #b21e2f (darker red)

## Layout Optimizations

### Spacing & Structure
- **Line height**: Increased to 1.2 for better readability
- **Stats boxes**: Added black borders for clear separation
- **Margins**: Optimized for maximum content space while maintaining readability
- **Content hierarchy**: Clear visual separation between sections

### Print Optimizations
- **High contrast**: All elements forced to high contrast for printing
- **Font weights**: All print text made bold or semi-bold
- **Clear borders**: Black borders on all cards for easy cutting
- **Color retention**: Print-color-adjust: exact for consistent output

## Content Improvements

### Abbreviated but Clear Text
- **Maintained full readability** while condensing text for space
- **Bold keywords** for quick scanning
- **Clear hierarchical structure** with consistent formatting
- **Essential information prioritized** in limited space

### Icon Usage
- **Meaningful icons** retained for quick visual reference
- **Larger emoji size** (14px) for tracking elements
- **Consistent iconography** throughout

## Screen vs Print Differences

### Screen Display
- **1.5x scaling** for easier editing and viewing
- **Larger spacing** between cards for comfortable browsing
- **Hover effects** maintained for interactivity

### Print Output
- **Exact MTG dimensions** (2.5" × 3.5")
- **Maximum contrast** for clear printing
- **Optimized font sizes** for small format readability
- **9 cards per page** in efficient 3×3 grid

## Recommendations for Use

### Printing Tips
1. **Use high-quality printer** with fresh ink/toner
2. **Print on bright white cardstock** (200-300 GSM)
3. **Use highest print quality** settings
4. **Ensure 100% scale** (no fit-to-page)
5. **Consider professional printing** for best results

### Usage Tips
1. **Good lighting** when using cards
2. **Consider magnifying glass** for detailed text if needed
3. **Organize by category** using the built-in filtering
4. **Print only needed cards** to reduce clutter
